# 🛡️ CyberShield Portfolio

A premium, cybersecurity-focused developer portfolio built with a **security-first aesthetic**. This project demonstrates a high-performance Single Page Application (SPA) built without frameworks, focusing on clean architecture, smooth UX, and cybersecurity-themed interactivity.

## ⚡ Key Highlights

### 🧠 Frontend Architecture (Vanilla JS SPA)
* **Framework-Free**: Built from scratch using Vanilla JavaScript.
* **Custom Navigation**: Implemented a `showPage()` system to handle section switching without page reloads.
* **Hash-Based Routing**: Uses URL hashes (`#home`, `#about`) for state management and browser history compatibility.
* **Lightweight Logic**: Direct DOM-based state handling for maximum efficiency.

---

### 🎯 Performance Optimization
* **Zero Dependencies**: Relies on zero external JavaScript libraries.
* **Efficient Rendering**: Minimal DOM reflows and optimized script execution.
* **Asset Optimization**: Uses CSS-generated grids and SVG noise overlays instead of heavy image files.
* **Viewport Awareness**: Leverages the Intersection Observer API to trigger animations only when visible.

---

### ✨ UI/UX Features
* **Modern Aesthetic**: Glassmorphism design utilizing CSS `backdrop-filter`.
* **Fluid Responsiveness**: Optimized for every device, from mobile touchscreens to 4K monitors.
* **Centralized Theming**: Uses CSS variables for consistent "Cyber-Green" branding across all components.
* **Atmospheric Visuals**: Animated cyber-grid background with a high-tech noise texture.

---

### 👁️ Interactions & Animations
* **Smart Skill Bars**: Animated progress indicators triggered by scroll position.
* **Terminal Simulation**: A dynamic command-line interface that cycles through security tools like `nmap` and `ClamAV`.
* **Dynamic Typing**: A recursive loop that cycles through professional roles with realistic character intervals.
* **Smooth Transitions**: CSS-driven `fadeUp` animations for seamless page entry.
